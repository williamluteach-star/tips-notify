# 快速開始指南

## 5分鐘快速設定

### 步驟1：安裝依賴
```bash
npm install
```

### 步驟2：複製環境變數檔案
```bash
cp .env.example .env
```

### 步驟3：填入必要的設定
編輯 `.env` 檔案，填入：
- LINE Channel Access Token
- LINE Channel Secret
- Google Sheets ID
- Google Service Account Email
- Google Private Key

詳細取得方式請參考 `設定指南.md`

### 步驟4：建立Google Sheets
1. 建立新的Google Sheets
2. 建立三個工作表：「學生資料表」、「作業記錄表」、「作業模板表」
3. 設定標題列（參考 `設定指南.md`）
4. 將服務帳號加入為編輯者

### 步驟5：測試連線
```bash
node scripts/test-connection.js
```

### 步驟6：啟動伺服器
```bash
npm start
```

### 步驟7：測試記錄作業
使用Postman或curl：
```bash
curl -X POST http://localhost:3000/api/homework \
  -H "Content-Type: application/json" \
  -d '{
    "studentName": "測試學生",
    "homeworkItem": "測試作業",
    "operator": "測試人員"
  }'
```

## 下一步

1. **匯入學生資料**：使用 `scripts/import-students.js` 或直接在Google Sheets輸入
2. **取得家長LINE User ID**：請家長加官方帳號為好友，從Webhook事件中取得
3. **開始使用**：工讀生可以在Google Sheets中直接記錄作業
