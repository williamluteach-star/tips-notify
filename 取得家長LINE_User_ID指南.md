# 取得家長 LINE User ID 指南

## 📋 方法總覽

有幾種方式可以取得家長的 LINE User ID，我們會提供最適合您的方法。

---

## 方法一：透過 Webhook 事件（推薦）

### 適用情況
- 系統已部署並設定 Webhook URL
- 家長加 LINE 官方帳號為好友時自動取得

### 步驟

1. **設定 Webhook URL**
   - 在 LINE Developers Console 設定 Webhook URL
   - 例如：`https://your-domain.com/webhook`

2. **請家長加好友**
   - 請家長搜尋並加入您的 LINE 官方帳號

3. **查看 Webhook 事件**
   - 當家長加好友時，系統會收到 `follow` 事件
   - 從事件中取得 `event.source.userId`

4. **填入學生資料表**
   - 在 Google Sheets「學生資料表」中
   - 找到對應的學生
   - 將 User ID 填入「家長LINE ID」欄位

---

## 方法二：從 LINE 官方帳號管理後台

### 適用情況
- 家長已經加好友
- 需要批量查看所有好友的 User ID

### 步驟

1. **登入 LINE 官方帳號管理後台**
   - 前往：https://manager.line.biz/
   - 使用您的 LINE 帳號登入

2. **查看好友列表**
   - 在左側選單選擇「好友」
   - 可以看到所有已加好友的使用者

3. **取得 User ID**
   - 點擊使用者可以查看詳細資訊
   - 或使用開發者工具查看（需要技術知識）

---

## 方法三：使用 LINE Messaging API 查詢（程式化）

### 適用情況
- 需要批量處理
- 有技術背景

### 實作方式

系統可以建立一個查詢功能，讓您輸入家長姓名或電話，系統自動查詢對應的 User ID。

---

## 方法四：請家長提供（最簡單）

### 適用情況
- 家長願意配合
- 需要快速取得

### 步驟

1. **請家長在 LINE 中發送訊息給官方帳號**
   - 例如：「我是XXX的家長」

2. **從 Webhook 事件中取得**
   - 系統會收到 `message` 事件
   - 從 `event.source.userId` 取得 User ID

3. **手動填入**
   - 在 Google Sheets 中找到對應學生
   - 填入 User ID

---

## 📝 在 Google Sheets 中填入 User ID

### 格式要求

- **欄位位置**：C 欄（「家長LINE ID」）
- **格式**：純文字，例如：`U1234567890abcdef1234567890abcdef`
- **注意**：不要包含空格或特殊符號

### 操作步驟

1. 開啟 Google Sheets「學生資料表」
2. 找到對應的學生行
3. 在 C 欄（家長LINE ID）輸入 User ID
4. 儲存

## 💡 建議流程

### 推薦做法：

1. **先匯入學生基本資料**（您正在做）
   - 學生姓名、年級、家長姓名、聯絡電話

2. **請家長加 LINE 官方帳號為好友**
   - 可以透過簡訊、Email 或紙本通知

3. **取得 User ID**
   - 使用上述任一方法取得

4. **填入 Google Sheets**
   - 在「學生資料表」中更新「家長LINE ID」欄位

5. **測試通知**
   - 記錄一筆作業測試通知是否正常

---

## ⚠️ 注意事項

1. **User ID 格式**
   - 通常是 `U` 開頭的長字串
   - 例如：`U1234567890abcdef1234567890abcdef`

2. **隱私保護**
   - User ID 是敏感資訊，請妥善保管
   - 不要分享給未授權的人員

3. **批量處理**
   - 如果有 100 位學生，建議分批處理
   - 可以先處理部分學生測試系統
