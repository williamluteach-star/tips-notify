# 系統畫面預覽說明

## Web管理介面

### 主要功能區域

#### 1. 記錄作業完成（左上）
- **學生姓名**：輸入框，必填
- **作業項目**：輸入框，必填
- **完成時間**：日期時間選擇器（預設為現在）
- **操作人員**：輸入框（預設為「系統」）
- **備註**：文字區域（選填）
- **記錄並發送通知**按鈕：提交表單

#### 2. 批量記錄作業（右上）
- **批量輸入框**：多行文字區域
  - 格式：每行一筆記錄
  - 格式：`學生姓名,作業項目,完成時間(選填),操作人員(選填)`
- **批量提交**按鈕：處理所有記錄

#### 3. 系統功能（左下）
- **發送今日摘要**按鈕：發送今日作業摘要給所有家長
- **重新整理**按鈕：重新載入資料

#### 4. 最近記錄（右下）
- 顯示最近的作業記錄列表
- 包含：學生姓名、作業項目、完成時間、操作人員、通知狀態

## 設計特色

### 視覺設計
- **漸層背景**：紫色漸層（#667eea 到 #764ba2）
- **白色卡片**：圓角設計，帶陰影效果
- **懸停效果**：卡片和按鈕有懸停動畫
- **響應式設計**：支援手機、平板、電腦

### 顏色系統
- **主色調**：紫色（#667eea）
- **成功**：綠色（#4CAF50）
- **資訊**：藍色（#2196F3）
- **錯誤**：紅色（#f8d7da）

### 互動體驗
- **即時回饋**：操作後立即顯示成功/錯誤訊息
- **自動重置**：成功提交後表單自動清空
- **載入狀態**：處理中顯示載入訊息

## 如何查看畫面

### 方法一：啟動本地伺服器
```bash
npm install
npm start
```
然後在瀏覽器開啟：`http://localhost:3000`

### 方法二：使用預覽工具
如果還沒有設定環境變數，介面仍可顯示，但API功能需要完整設定後才能使用。

## LINE訊息畫面

### 家長收到的通知訊息格式

#### 即時通知：
```
【作業完成通知】

張小明已完成以下作業：
📝 數學練習本P.10
⏰ 完成時間：2024年01月15日 14:00

感謝您的關注！
```

#### 每日摘要：
```
【2024年01月15日 作業完成摘要】

張小明今日完成：

1. 數學練習本P.10
   ⏰ 2024-01-15 14:00

2. 國文作文
   ⏰ 2024-01-15 16:30

共完成 2 項作業

感謝您的關注！
```

#### 家長查詢回覆：
```
【近期作業完成記錄】

1. 數學練習本P.10
   完成時間：2024-01-15 14:00

2. 國文作文
   完成時間：2024-01-15 16:30
```

## 使用流程畫面

1. **工讀生操作**：
   - 開啟Web管理介面
   - 輸入學生姓名和作業項目
   - 點擊「記錄並發送通知」
   - 看到成功訊息

2. **系統處理**：
   - 記錄寫入Google Sheets
   - 自動發送LINE通知給家長

3. **家長接收**：
   - 在LINE中收到通知訊息
   - 可回覆「查詢」查看記錄
