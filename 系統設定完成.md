## 📝 下一步操作

### 1. 匯入學生資料

#### 方式一：使用 CSV 匯入
```bash
node scripts/import-students.js scripts/example-students.csv
```

#### 方式二：直接在 Google Sheets 輸入
- 開啟「學生資料表」
- 從第 2 行開始輸入學生資料
- 格式：`學生姓名 | 年級 | 家長LINE ID | 家長姓名 | 聯絡電話 | 備註`

### 2. 取得家長 LINE User ID

#### 方法一：透過 Webhook（推薦）
1. 部署系統並設定 Webhook URL
2. 請家長加 LINE 官方帳號為好友
3. 從 Webhook 事件中取得 `event.source.userId`
4. 填入「學生資料表」的「家長LINE ID」欄位

#### 方法二：從 LINE 官方帳號管理後台
1. 登入 LINE 官方帳號管理後台
2. 在「好友」列表中查看 User ID

### 3. 開始使用

1. **啟動伺服器**（如果還沒啟動）
   ```bash
   npm start
   ```

2. **開啟 Web 管理介面**
   - 網址：http://localhost:3000

3. **記錄作業**
   - 輸入學生姓名和作業項目
   - 系統會自動記錄到 Google Sheets
   - 並發送 LINE 通知給家長
