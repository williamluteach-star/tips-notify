# 使用說明 - 給工讀生

## 快速開始

### 方式一：使用Google Sheets直接輸入（推薦）

1. 開啟 Google Sheets（作業記錄表）
2. 在最後一行新增記錄：
   - **時間戳記**：自動填入（或手動輸入）
   - **學生姓名**：輸入學生姓名
   - **作業項目**：輸入作業名稱
   - **完成時間**：輸入完成時間（格式：YYYY-MM-DD HH:mm）
   - **操作人員**：輸入您的名字
   - **通知狀態**：系統會自動更新
   - **備註**：（選填）

3. 系統會自動發送通知給家長

### 方式二：使用API（適合批量輸入）

#### 使用Postman或類似工具

**單筆記錄：**
```
POST https://your-server.com/api/homework
Content-Type: application/json

{
  "studentName": "張小明",
  "homeworkItem": "數學練習本P.10",
  "completedTime": "2024-01-15T14:00:00",
  "operator": "工讀生A"
}
```

**批量記錄：**
```
POST https://your-server.com/api/homework/batch
Content-Type: application/json

{
  "records": [
    {
      "studentName": "張小明",
      "homeworkItem": "數學練習本P.10",
      "operator": "工讀生A"
    },
    {
      "studentName": "李小花",
      "homeworkItem": "國文作文",
      "operator": "工讀生A"
    }
  ]
}
```

## 日常操作流程

### 早上
1. 檢查前一天的作業記錄是否完整
2. 如有遺漏，補登記錄

### 作業時間
1. 學生完成作業後，立即在Google Sheets記錄
2. 或使用API批量記錄

### 晚上
1. 發送每日摘要給所有家長（可選）
2. 檢查通知狀態，確認所有家長都收到通知

## 常見問題

### Q: 如何確認家長是否收到通知？
A: 檢查Google Sheets中「通知狀態」欄位，應顯示「已通知」

### Q: 如果輸入錯誤怎麼辦？
A: 直接在Google Sheets中修改該筆記錄即可

### Q: 如何查詢某位學生的所有作業記錄？
A: 在Google Sheets中使用篩選功能，選擇「學生姓名」欄位

### Q: 家長說沒收到通知怎麼辦？
A: 
1. 檢查「學生資料表」中的「家長LINE ID」是否正確
2. 確認家長已加LINE官方帳號為好友
3. 檢查LINE訊息額度是否用完

### Q: 可以一次記錄多筆作業嗎？
A: 可以！使用批量記錄API，或直接在Google Sheets中複製貼上多筆記錄

## 注意事項

1. **學生姓名必須完全一致**：請確保輸入的學生姓名與「學生資料表」中的完全一致
2. **時間格式**：完成時間請使用 `YYYY-MM-DD HH:mm` 格式，例如：`2024-01-15 14:30`
3. **不要刪除記錄**：如需修改，請直接編輯，不要刪除整行
4. **定期備份**：Google Sheets會自動儲存，但建議定期下載備份
