# 取得已加好友家長的 LINE User ID

## 📋 情況說明

家長已經加 LINE 官方帳號為好友，現在需要取得他們的 User ID 並填入學生資料表。

---

## 方法一：請家長發送訊息（最簡單）

### 步驟

1. **請家長在 LINE 中發送訊息給官方帳號**
   - 訊息內容：例如「我是XXX的家長」或「我是張小明的媽媽」
   - 這樣系統就能知道是哪位家長

2. **查看終端機輸出**
   - 當家長發送訊息時，終端機會自動顯示：
     ```
     💬 收到家長訊息
        User ID: U1234567890...
        訊息內容: 我是張小明的媽媽
     ```

3. **配對並填入**
   - 根據訊息內容（學生姓名）找到對應的學生
   - 將 User ID 填入「學生資料表」的 C 欄

### 優點
- ✅ 簡單直接
- ✅ 可以確認家長身份
- ✅ 自動顯示 User ID

---

## 方法二：使用 LINE 官方帳號管理後台

### 步驟

1. **登入管理後台**
   - 前往：https://manager.line.biz/
   - 使用您的 LINE 帳號登入

2. **查看好友列表**
   - 在左側選單選擇「好友」
   - 可以看到所有已加好友的使用者

3. **取得 User ID**
   - 點擊使用者可以查看詳細資訊
   - 或使用瀏覽器開發者工具查看（需要技術知識）

### 注意
- 管理後台可能不會直接顯示 User ID
- 需要透過技術方式取得
