# 🚀 電子通知系統 — Railway 雲端部署指南

部署完成後，系統會 **24 小時在線**，不需要每次開電腦。

---

## 第一步：把程式碼推上 GitHub

### 1-1 安裝 Git（如果還沒安裝）
```bash
brew install git
```

### 1-2 在 GitHub 建立新 repo
1. 打開 https://github.com → 登入（或先註冊免費帳號）
2. 點右上角「+」→「New repository」
3. 名稱輸入：`tips-notify`
4. 選 **Private**（私有，保護您的金鑰）
5. 點「Create repository」

### 1-3 上傳程式碼
在 Terminal 輸入（把 `YOUR_USERNAME` 換成您的 GitHub 帳號）：

```bash
cd ~/電子通知
git init
git add .
git commit -m "初始版本"
git remote add origin https://github.com/YOUR_USERNAME/tips-notify.git
git push -u origin main
```

---

## 第二步：在 Railway 部署

### 2-1 建立 Railway 帳號
打開 https://railway.app → 用 GitHub 帳號登入

### 2-2 建立新專案
1. 點「New Project」
2. 選「Deploy from GitHub repo」
3. 選剛才建立的 `tips-notify`
4. Railway 會自動偵測 Node.js 並開始部署

### 2-3 設定環境變數（最重要！）
點進專案 → 點選您的服務 → 上方選「**Variables**」頁籤

逐一新增以下變數（值從您電腦的 `.env` 檔複製）：

| 變數名稱 | 說明 |
|---------|------|
| `LINE_CHANNEL_ACCESS_TOKEN` | 從 .env 複製 |
| `LINE_CHANNEL_SECRET` | 從 .env 複製 |
| `GOOGLE_SHEETS_ID` | 從 .env 複製 |
| `GOOGLE_SERVICE_ACCOUNT_EMAIL` | 從 .env 複製 |
| `LINE_BASIC_ID` | `@334tjghl` |
| `NODE_ENV` | `production` |

### ⚠️ GOOGLE_PRIVATE_KEY 特別注意
這個值包含多行，貼上時要這樣做：
1. 打開 .env 檔，複製 `"-----BEGIN PRIVATE KEY-----` 到 `-----END PRIVATE KEY-----\n"` 整段（含引號）
2. 在 Railway Variables 頁面點「RAW Editor」
3. 把整段（去掉首尾引號）貼進去
4. Railway 會自動保留換行

---

## 第三步：取得固定網址

1. 點選您的 Railway 服務
2. 上方選「**Settings**」頁籤
3. 找到「**Domains**」→ 點「Generate Domain」
4. 會得到像這樣的網址：
   ```
   https://tips-notify-production.up.railway.app
   ```
5. 記下這個網址

---

## 第四步：設定 LINE Webhook

1. 打開 https://developers.line.biz/console/
2. 選您的 Messaging API Channel
3. 找到「**Webhook URL**」
4. 填入：
   ```
   https://tips-notify-production.up.railway.app/webhook
   ```
5. 開啟「**Use webhook**」
6. 點「**Verify**」→ 應該看到「Success」✅

---

## 第五步：確認系統正常

打開瀏覽器連到您的 Railway 網址，例如：
```
https://tips-notify-production.up.railway.app
```

應該看到作業通知系統的 Web 介面。

---

## 完成！運作方式

```
家長傳訊息給 LINE OA
       ↓
Railway 伺服器收到 Webhook
       ↓
User ID 記錄 + 家長配對
       ↓
老師在 Web 介面記錄作業
       ↓
Google Sheets 寫入 + LINE 通知家長
```

## 日後更新程式碼

每次修改程式後，在 Terminal 執行：
```bash
cd ~/電子通知
git add .
git commit -m "更新內容說明"
git push
```

Railway 會自動偵測到 push 並重新部署（約 1-2 分鐘）。
